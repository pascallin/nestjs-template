# A Nest Template

## Tech Stack

- [Nest](https://github.com/nestjs/nest) framework.

## Installation

```bash
$ npm install # or yarn
$ npm run prepare # prepare husky
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
